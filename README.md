# TimeBridge — Time Zone Dashboard

A polished multilingual time-zone comparison dashboard for long-distance couples, overseas friends, and global workers.

## Features

- Korean / English / Chinese language switcher
- Home and target time-zone selector
- Live current-time comparison
- Specific date/time conversion
- Availability-window conversion
- Presets for Seoul ↔ New York, London, Shanghai, and Los Angeles
- Pure static HTML/CSS/JS; no build step required

## Design reference

The UI references the design-system notes from [VoltAgent/awesome-design-md](https://github.com/VoltAgent/awesome-design-md), especially the Linear/Stripe/Superhuman-inspired dashboard language: dark premium surfaces, fine hairline borders, atmospheric gradients, and polished SaaS controls.
